Progressive Web App

This is my first Progressive Web app with React and serverless(AWS)

Objective

To build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events.

Through the project

In this project I focused on learning test driven development, also solidifying my React knowledge. At first I created user story, then coded tests and functions. Through this project I used several tests (unit, integration, user acceptance, end-to-end) and they helped me constructing codes, finding an error and check which part is not working.
In order to get used to test-driven-environment, I tried to use tests as much as possible rather than console.log and developer tool, which was a challenge for me but later it helped me to detect which code causes an error.
Also learning serverless and Google API was a whole new experience how serverless works and authenticates user, then get information from Google API.

Next step

At the beginning it was a big challenge with all information about tests and React, and serverless. However, I believe I grew up much more than before and became more confident with React and I would love to keep coding with React and create progressive web app.

Tools used

  • Jest
  • Jest-Cucumber
  • Puppeteer
