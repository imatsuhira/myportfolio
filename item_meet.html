Progressive Web App

This is my first Progressive Web app with React and serverless(AWS)

To build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events.

In this project I learned variety of tests(unit test, integration test, user acceptance testing, end to end testing) helped me finding an error and check which part of codes is not working.
In order to get used to test-driven-environment, I tried to use tests as much as possible rather than console.log and devtool, which wa a challenge for me but later it helped me to detect which code causes an error.
Also learning serverless and Google API was a whole new experience. How serverless works and authenticates user, then get information from Google API was a lot to absorb and I will definitely keep building apps with these.
